# Bank-Management-System
Full-stack Bank Management System developed using Spring Boot and MySQL with secure authentication, transaction management, and role-based authorization.
The project includes separate User and Admin dashboards with secure authentication and role-based authorization.

# Features:

 User Features 
. User Registration & Login
. Secure Authentication
. Dashboard Overview
. Check Account Balance
. Transfer Money
. Transaction History
. Add Money via Card

Admin Features
. Admin Dashboard
. User Management
. Account Monitoring
. Transaction Monitoring
. Block / Delete Users
. Search Users

# Tech Stack
. Java
. Spring Boot
. Spring Security
. Hibernate / JPA
. MySQL
. Thymeleaf
. Tailwind CSS
. Maven
. Security
. Authentication & Authorization using Spring Security
. BCrypt Password Encryption
. Role-Based Access Control
. Secure Login System
. Database
